# DocMind — Technical Document RAG System

A Streamlit-based RAG application for querying technical PDFs.

## Features

- PDF ingestion & chunking
- Semantic search using FAISS
- Zero-shot classification for document validation
- Grounded answer generation using FLAN-T5
- Source attribution

## Tech Stack

- LangChain
- FAISS
- HuggingFace Transformers
- Streamlit
